Arma3Spotter
============

![](http://img.shields.io/travis/KoffeinFlummi/Arma3Spotter.svg)


### Setup

**Requires NodeJS, NPM, Bower and Grunt.**

```shell
cd src
npm install
bower install
grunt serve
```

### Building App Packages

*Requires OSX & XCode & XCode Command Line Tools*

```shell
cd src
grunt build
grunt cordovaBuild
```
